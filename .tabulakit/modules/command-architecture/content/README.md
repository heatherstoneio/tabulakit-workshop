# Command Architecture

A practical system for handling complexity under pressure. Military organizational principles translated for any organization where leaders face overwhelming demands on their attention and decision-making.

## The Problem It Solves

If you're a leader feeling overwhelmed, it's probably not because you lack intelligence or work ethic. It's because you're trying to hold everything in your head at once — personnel, intelligence, operations, resources, plans, communications, and stakeholder narratives — all competing for the same mental bandwidth.

The military solved this centuries ago. Complexity becomes manageable when you divide it across two dimensions: **domains** (horizontal specialization) and **altitudes** (vertical hierarchy of decision-making).

## The Framework

| Component | What It Does |
|-----------|-------------|
| [Seven Domains (S1-S7)](quick-reference.md) | Partition all organizational concerns into distinct areas |
| [Three Altitudes](quick-reference.md) | Separate strategic (why), operational (how), and tactical (what) thinking |
| [Planning Process](planning-process.md) | Structured planning that produces actionable orders |
| [Orders Sequence](orders-communication.md) | Communication discipline: warn → brief → update |
| [Execution Framework](execution-framework.md) | Bridge from plan to completion with feedback loops |
| [Operating Rhythm](operating-rhythm.md) | Cadences that keep everything running |

## How to Use This

1. **Start with the [Quick Reference](quick-reference.md)** — one-page overview of all components
2. **When planning:** Follow the [Planning Process](planning-process.md)
3. **When executing:** Use the [Execution Framework](execution-framework.md)
4. **When communicating:** Follow the [Orders Sequence](orders-communication.md)
5. **For ongoing operations:** Establish an [Operating Rhythm](operating-rhythm.md)

## Key Principle

> **Intent over control.** Communicate the *why* and the desired end state. Let people figure out the *how*. When conditions change, people who understand intent can adapt without waiting for new instructions.
