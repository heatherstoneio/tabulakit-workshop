# Pre-Work Checklist

Complete these steps **before** {{course_date}} to be ready for the hands-on exercises.

## Required

- [ ] **Create a GitHub account** at [github.com](https://github.com) (if you don't have one)
- [ ] **Install a text editor** — [VS Code](https://code.visualstudio.com/) recommended
- [ ] **Install Claude Code** — see [claude.ai/code](https://claude.ai/code) for setup instructions

## Recommended

- [ ] **Fork the TabulaKit template** — click "Use this template" on the [TabulaKit repo](https://github.com/heatherstoneio/tabulakit)
- [ ] **Clone your fork** to your computer
- [ ] **Preview locally** — run `npx live-server site --port=3000` to see the default site

## Verify Your Setup

If everything is working, you should be able to:

1. Open your forked repo in VS Code
2. Launch Claude Code
3. Type `/startup` and see the setup wizard begin

If you run into issues, don't worry — we'll troubleshoot at the start of the workshop.

## Need Help?

Contact {{instructor_name}} before the course if you're stuck on any setup step.
