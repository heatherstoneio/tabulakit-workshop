# Pre-Work Checklist

Complete these steps at least 48 hours before the workshop. They take about 10-15 minutes. If you get stuck on any step, just set it aside — we'll have a troubleshooting window at the start of the workshop.

## 1. Create a GitHub Account

Go to [github.com/signup](https://github.com/signup) and create a free account.

GitHub stores your project and tracks every change you make — think of it like a smart filing cabinet in the cloud that you own.

## 2. Install VS Code

Download and install [Visual Studio Code](https://code.visualstudio.com/).

This is the application where you'll do your work — you'll type, talk to the AI, and see your changes here.

**Windows users — one extra step:**
Open PowerShell (search in Start menu, right-click, "Run as Administrator") and paste:
```
wsl --install
```
Restart your computer when prompted. Then open VS Code and install the **WSL** extension (click the square icon on the left sidebar, search "WSL", install it).

## 3. Sign Up for Claude Pro

Go to [claude.ai](https://claude.ai) and create an account, then upgrade to Claude Pro ($20/month).

This is the AI that powers everything we'll build in the workshop. The Pro subscription gives you enough usage for the workshop and ongoing work.

## 4. Install the Claude Code Extension

In VS Code, click the Extensions icon (square on the left sidebar), search for **Claude Code**, and install it.

## 5. Install the TabulaKit Extension

This extension adds a dashboard to VS Code with a live site preview and helpful controls.

1. Download the extension file: [TabulaKit v0.1.0](https://github.com/heatherstoneio/tabulakit-vscode/releases/download/v0.1.0/tabulakit-0.1.0.vsix)
2. In VS Code, open the Extensions panel (square icon on the left sidebar)
3. Click the **"..."** menu at the top of the Extensions panel
4. Select **Install from VSIX...**
5. Choose the file you just downloaded

> **Note:** We're working on publishing TabulaKit to the VS Code Marketplace. Once it's there, you'll be able to search and install it directly — no download needed.

## 6. Verify It Works

Open VS Code, find Claude Code in the bottom panel, and try typing "hello". If the AI responds, you're all set.

## 7. Think About What to Build

This is the most important prep you can do:

- **What's the one thing** in your business that would benefit most from getting organized?
- **What materials do you already have?** Documents, spreadsheets, notes, emails — anything relevant. Have these accessible on your laptop during the workshop.
- **Who needs access?** Just you? Your team? Clients? This affects which setup options you'll choose.

You don't need all the answers. Just come with a direction and whatever existing materials you have.

## Trouble?

If you hit a wall on any step, don't stress. We'll have time at the start of the workshop to troubleshoot. The most important thing is showing up with a laptop and a direction in mind.
