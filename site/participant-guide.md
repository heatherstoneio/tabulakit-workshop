# Participant Guide

## What to Expect

This is a hands-on workshop. You'll spend most of the time building and customizing your own documentation site. Come prepared to:

- **Experiment** — try things, break things, fix things
- **Ask questions** — no question is too basic
- **Help each other** — share what you discover

## What You'll Need

- A laptop with internet access
- A GitHub account ([sign up here](https://github.com/signup))
- A text editor (VS Code recommended)
- Claude Code installed ([setup guide](https://claude.ai/code))

See the full [pre-work checklist](pre-work.md).

## What You'll Walk Away With

- A live documentation site you built yourself
- Understanding of how markdown-based documentation works
- Experience using AI-assisted development tools
- A template you can reuse for future projects

## During the Workshop

- **Follow along** with the exercises at your own pace
- **Ask for help** if you get stuck — raise your hand or ask Claude Code
- **Take notes** — this site is yours to keep and modify

## After the Workshop

Your site is yours. Keep building on it:

- Add more pages and content
- Try different deployment targets
- Share it with your team
- Use it as a template for your next project
